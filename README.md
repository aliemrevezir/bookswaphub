# 📚 BookSwap Hub

A modern, user-friendly platform designed for book enthusiasts to easily exchange books. Built with React and Tailwind CSS, featuring a beautiful dark mode and responsive design.

![BookSwap Hub](https://wezirim.com/bookswaphub.png)

## ✨ Features

- [ ] User Authentication (Register/Login)
  - Secure authentication system
  - Password recovery
  - Profile management
- [ ] Book Management
  - List books for swapping
  - Book search and discovery
  - Book condition tracking
- [ ] Swap System
  - Request book swaps
  - Accept/reject swap requests
  - Swap history
- [ ] User Interaction
  - Real-time messaging
  - Rating and review system
  - User reputation system
- [x] Modern UI/UX
  - Responsive design
  - Dark mode support
  - Accessible components
  - Smooth animations

## 🛠 Tech Stack

- **Frontend**
  - React 18
  - Tailwind CSS
  - React Router v6
  - Headless UI
  - Hero Icons
- **Backend** (Coming Soon)
  - Node.js
  - Express.js
  - PostgreSQL
  - JWT Authentication

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/bookswaphub.git
   cd bookswaphub
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser
